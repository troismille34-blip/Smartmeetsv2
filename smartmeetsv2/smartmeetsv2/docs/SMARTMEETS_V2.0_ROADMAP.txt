1.1 LANGUAGE DETECTION & DIALECT RECOGNITION
────────────────────────────────────────────

Problem: Web Speech API erkennt Dialekte nicht, nur Standard-Sprachen

Lösung Stack:
  
  Option A: OpenAI Whisper (EMPFOHLEN)
    ✅ Beste Accuracy für Dialekte
    ✅ Unterstützt 99 Sprachen + Dialekte
    ✅ Bern-Deutsch erkennung
    ✅ Kosteneffektiv: $0.006/min
    ✅ Multilingual in einer API
    ✅ Timestamps & Confidence-Scores
    
    API Implementierung:
    ```javascript
    const transcribeAudio = async (audioBlob) => {
      const formData = new FormData();
      formData.append('file', audioBlob);
      formData.append('model', 'whisper-1');
      formData.append('language', 'de'); // Auto-detect Dialekt
      
      const response = await fetch('https://api.openai.com/v1/audio/transcriptions', {
        method: 'POST',
        headers: { 'Authorization': `Bearer ${OPENAI_API_KEY}` },
        body: formData
      });
      
      return response.json();
    };
    ```

  Option B: Google Cloud Speech-to-Text (Backup)
    ✓ Auch sehr gut
    ✓ Streaming-Support
    ✓ Kostenlich höher: $0.024/min
    ✓ Enterprise-Support
    
  Option C: Azure Speech Services (für Teams-Integration)
    ✓ Native Teams-Integration
    ✓ Custom Model Training
    ✓ Hohe Kosten

Recommendation: **OpenAI Whisper + Google Cloud als Fallback**

1.3 DIALECT HANDLING - BERN DEUTSCH
─────────────────────────────────────

Bern-Deutsch Spezifika:
  - "Grüezi" statt "Guten Tag"
  - "Schön" statt "Schön"
  - Spezifische Vokabeln
  - Schnellere Sprechgeschwindigkeit
  - Nasallaute

Umsetzung:

  A) Whisper Language-Hint:
     ```javascript
     // Sag OpenAI dass es Schweizerdeutsch ist
     const response = await transcribeWithHints({
       audioBlob,
       language: 'de',
       context: 'Swiss Business Meeting', // Hilft Whisper
       glossary: {
         'Grüezi': 'Grüezi',
         'Chöschtli': 'Chöschtli' // Spezialwörter
       }
     });
     ```

  B) Post-Processing (Regex-Fixes):
     ```javascript
     const swissGermanFixes = {
       'guten tag': 'Grüezi',
       'danke viel': 'Merci villmal',
       // ... weitere Patterns
     };
     
     let transcript = result.text;
     Object.entries(swissGermanFixes).forEach(([from, to]) => {
       transcript = transcript.replace(new RegExp(from, 'gi'), to);
     });
     ```

  C) Custom Model Fine-Tuning (Advanced):
     - Trainiere OpenAI Whisper mit Bern-Deutsch Samples
     - Verbessert Accuracy um 15-20%
     - Kostet: ~$500-1000 Training

Recommendation: **A + B kombinieren für v2.0**

5.1 VIDEO RECORDING IMPLEMENTATION
───────────────────────────────────

Upgrade von Audio zu Video:

```javascript
const useVideoRecorder = () => {
  const [isRecording, setIsRecording] = useState(false);
  const [videoBlob, setVideoBlob] = useState(null);
  const canvasRef = useRef(null);
  const mediaRecorderRef = useRef(null);
  
  const startVideoRecording = async () => {
    try {
      const stream = await navigator.mediaDevices.getDisplayMedia({
        video: {
          cursor: 'always' // Zeige Cursor/Bildschirm
        },
        audio: true
      });
      
      // Mit WebRTC verbinden
      const mediaRecorder = new MediaRecorder(stream, {
        mimeType: 'video/webm;codecs=vp9,opus'
      });
      
      const chunks = [];
      mediaRecorder.ondataavailable = (e) => chunks.push(e.data);
      mediaRecorder.onstop = () => {
        const blob = new Blob(chunks, { type: 'video/webm' });
        setVideoBlob(blob);
      };
      
      mediaRecorder.start();
      setIsRecording(true);
    } catch (error) {
      console.error('Screen capture denied:', error);
    }
  };
  
  return { isRecording, videoBlob, startVideoRecording };
};
```

5.2 VIDEO PLAYBACK & SCRUBBING
───────────────────────────────

Mit synchronisiertem Transkript:

```javascript
const VideoPlayer = ({ videoBlob, transcript }) => {
  const [currentTime, setCurrentTime] = useState(0);
  const [isPlaying, setIsPlaying] = useState(false);
  const videoRef = useRef(null);
  
  const currentSegments = transcript.filter(seg => {
    const [mins, secs] = seg.timestamp.split(':').map(Number);
    const segmentTime = mins * 60 + secs;
    return Math.abs(currentTime - segmentTime) < 5;
  });
  
  return (
    <div style={{ display: 'flex', gap: '16px' }}>
      {/* Video */}
      <div style={{ flex: 2 }}>
        <video
          ref={videoRef}
          src={URL.createObjectURL(videoBlob)}
          onTimeUpdate={() => setCurrentTime(videoRef.current.currentTime)}
          controls
          style={{ width: '100%' }}
        />
      </div>
      
      {/* Synchronized Transcript */}
      <div style={{ flex: 1, overflowY: 'auto' }}>
        {currentSegments.map(seg => (
          <div key={seg.timestamp} style={{
            padding: '12px',
            background: '#f0f0f0',
            borderRadius: '8px',
            marginBottom: '8px'
          }}>
            <strong>{seg.speaker}</strong>
            <p>{seg.text}</p>
            <small>{seg.timestamp}</small>
          </div>
        ))}
      </div>
    </div>
  );
};
```

8.3 DATABASE SCHEMA
────────────────────

```sql
-- Users
CREATE TABLE users (
  id UUID PRIMARY KEY,
  email VARCHAR(255) UNIQUE,
  name VARCHAR(255),
  avatar_url TEXT,
  company_id UUID,
  role ENUM('admin', 'user'),
  created_at TIMESTAMP,
  updated_at TIMESTAMP
);

-- Meetings
CREATE TABLE meetings (
  id UUID PRIMARY KEY,
  title VARCHAR(255),
  description TEXT,
  organizer_id UUID REFERENCES users,
  start_time TIMESTAMP,
  end_time TIMESTAMP,
  status ENUM('scheduled', 'recording', 'completed', 'cancelled'),
  recording_url TEXT,
  video_url TEXT,
  participant_count INT,
  created_at TIMESTAMP,
  updated_at TIMESTAMP
);

-- Meeting Participants
CREATE TABLE meeting_participants (
  id UUID PRIMARY KEY,
  meeting_id UUID REFERENCES meetings,
  user_id UUID REFERENCES users,
  email VARCHAR(255),
  name VARCHAR(255),
  join_time TIMESTAMP,
  leave_time TIMESTAMP,
  is_organizer BOOLEAN,
  speaker_profile_id UUID
);

-- Transcripts
CREATE TABLE transcripts (
  id UUID PRIMARY KEY,
  meeting_id UUID REFERENCES meetings,
  content TEXT,
  language VARCHAR(10),
  dialect VARCHAR(50),
  duration INT,
  created_at TIMESTAMP
);

-- Transcript Segments
CREATE TABLE transcript_segments (
  id UUID PRIMARY KEY,
  transcript_id UUID REFERENCES transcripts,
  speaker_id UUID REFERENCES meeting_participants,
  speaker_name VARCHAR(255),
  start_time INT,
  end_time INT,
  content TEXT,
  is_edited BOOLEAN,
  edited_by UUID REFERENCES users,
  confidence_score FLOAT
);

-- Action Items
CREATE TABLE action_items (
  id UUID PRIMARY KEY,
  meeting_id UUID REFERENCES meetings,
  task_description TEXT,
  assignee_id UUID REFERENCES users,
  due_date DATE,
  priority ENUM('high', 'medium', 'low'),
  status ENUM('open', 'in_progress', 'completed'),
  timestamp_in_meeting INT,
  created_at TIMESTAMP,
  updated_at TIMESTAMP
);

-- Speaker Profiles
CREATE TABLE speaker_profiles (
  id UUID PRIMARY KEY,
  user_id UUID REFERENCES users,
  voice_signature BYTEA,
  known_names JSONB,
  created_at TIMESTAMP,
  updated_at TIMESTAMP
);
```
